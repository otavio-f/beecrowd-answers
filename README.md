Answers to Beecrowd's (former URI) problems

# C

Beecrowd C99 uses the following `gcc` flags

`-std=c99 -O2 -lm`

In addition, all solutions were tested locally using the following flags:

`-Wall -Werror -std=c99 -O2 -lm`
